Module: DataConversionFactory

A factory function that returns a parser function based on the given source type. Currently, it supports "csv" but is designed to be extended to support additional data types.

In the future, all parsers returned by this factory will be expected to follow the same signature and return type as csvParser, ensuring uniformity across different data sources.

(async, static) csvParser(filePath) → {Promise.<Array.<Object>>}

Parses a CSV file and returns its contents as an array of objects. Each object represents a row in the CSV, with key-value pairs corresponding to column headers and their values.
